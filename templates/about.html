About

This app was inspired by this post by Dave Winer. Dave was looking for an XML-RPC service but it was much easier for me to create an simple RESTful service that either returns an JSON object with the image base64'd or just stream the image back in the request. The application is hosted on Google App Engine. Since this is an GAE application I'm using the Image API provided by Google instead of ImageMagick and/or GraphicsMagick

The code for this site is hosted on GitHub

BTW - I ripped off the styles for this site from the Django Admin stylesheet
